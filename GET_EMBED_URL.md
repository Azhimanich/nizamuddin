# Cara Mendapatkan Embed URL dari Google Maps Short URL

Untuk mendapatkan embed URL yang tepat dari link: https://maps.app.goo.gl/7nnUT6gDu319riVSA

## Langkah-langkah:

1. **Buka link di browser:**
   - Buka: https://maps.app.goo.gl/7nnUT6gDu319riVSA
   - Link akan redirect ke Google Maps dengan lokasi yang tepat

2. **Dapatkan Embed URL:**
   - Setelah peta terbuka, klik tombol **"Bagikan"** (Share) di sidebar kiri
   - Pilih tab **"Sematkan peta"** (Embed map)
   - Pilih ukuran peta (Kecil, Sedang, Besar, atau Kustom)
   - Salin kode HTML yang muncul, atau ambil URL dari atribut `src` pada tag `<iframe>`

3. **Contoh format embed URL:**
   ```
   https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3966...!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x2e69f...!2z...!5e0!3m2!1sid!2sid!4v1234567890!5m2!1sid!2sid
   ```

4. **Update di Admin Panel:**
   - Login ke admin panel
   - Buka menu **"Manajemen Kontak"**
   - Pilih tab **"Peta"**
   - Edit peta yang sudah ada
   - Paste embed URL ke field **"Google Maps Embed URL"**
   - Simpan

## Alternatif: Menggunakan Koordinat

Jika tidak bisa mendapatkan embed URL, Anda bisa menggunakan koordinat:
- Buka link di Google Maps
- Klik kanan pada marker lokasi
- Pilih "Apa yang ada di sini?"
- Koordinat akan muncul di bagian bawah
- Masukkan koordinat (Latitude, Longitude) di admin panel
